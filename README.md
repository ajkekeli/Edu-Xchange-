EduXchange | Learn. Share. Grow.
EduXchange is a student-powered web application designed to bridge the gap between students who have academic resources and those who need support. It provides a centralized hub for knowledge sharing, peer-to-peer tutoring, and collaborative learning.

 Features
User Authentication: Secure signup and login system with session management to protect student data.

Academic Feed: A real-time discussion forum where students can post questions and receive answers from their peers.

Resource Repository: A digital library allowing users to upload and download study materials such as PDFs and Word documents.

Private Assistance: A dedicated "Request-Assistance" feature for students to submit specific issues and receive private help via email.

Admin Dashboard: A comprehensive management panel for authorized users to oversee platform stats, users, questions, and resources.

Modern UI: Features a responsive design with a "Futuristic" dark mode toggle and smooth page transitions.

 Technical Stack
Frontend: HTML5, CSS3 (including custom themes), and JavaScript (ES6+).

Backend: PHP (API-driven architecture).

Database: MySQL (Relational schema with tables for users, resources, and discussion threads).

Asynchronous Operations: Uses the Fetch API for seamless data updates without page reloads.

 Database Structure
The system relies on a relational database (forum.sql) comprising the following key tables:

users: Stores student credentials and roles (User/Admin).

questions & replies: Powers the community discussion feed.

resources: Manages file paths and metadata for shared academic documents.

help_requests: Tracks private tutoring and assistance tickets.


How to Use


Get Started:
Create an account on the Signup page.

Collaborate: Use the Home feed to ask academic questions.

Share: Visit the Resources tab to upload your notes or download existing materials.

Seek Help: Use the Request-Assistance form to get one-on-one help with specific topics.

Moderation & Security
EduXchange is built as a safe and moderated space. Admins have exclusive access to the Admin Panel to monitor activity and ensure a productive learning environment.
